## 学习路线图 (12周计划)

### 第一阶段: 基础夯实 (第1-3周)

#### 模块1: Python 进阶 (1周)

- [ ] Python 异步编程 (asyncio)
- [ ] Python 类型系统 (Type Hints)
- [ ] 常用库: pydantic, httpx, aiohttp
- [ ] **实战**: 用 Python 写一个简单的 API 服务

#### 模块2: LLM 基础 (1周)

- [ ] 大语言模型原理入门
- [ ] OpenAI API 使用
- [ ] Prompt Engineering 基础
- [ ] **实战**: 实现一个智能对话机器人

#### 模块3: LangChain 核心 (1周)

- [ ] LangChain 架构概览
- [ ] Chain、Prompt、Model 组件
- [ ] Output Parser 输出解析
- [ ] **实战**: 构建一个文档问答系统

### 第二阶段: 核心技能 (第4-7周)

#### 模块4: RAG 检索增强生成 (2周)

- [ ] 向量数据库原理
- [ ] Embedding 模型选择
- [ ] 文档加载与分块
- [ ] 检索策略优化
- [ ] **实战**: 企业知识库问答系统

#### 模块5: Agent 智能体 (2周)

- [ ] Agent 架构设计
- [ ] Tools 工具调用
- [ ] Memory 记忆系统
- [ ] Multi-Agent 多智能体
- [ ] **实战**: 自动化办公助手

### 第三阶段: 工程化 (第8-10周)

#### 模块6: 后端工程 (2周)

- [ ] FastAPI 框架深入
- [ ] 流式响应 (SSE/WebSocket)
- [ ] 数据库设计 (PostgreSQL + 向量库)
- [ ] **实战**: AI 应用后端 API

#### 模块7: 前端集成 (1周)

- [ ] Next.js 集成 AI 能力
- [ ] 流式 UI 实现
- [ ] **实战**: AI 全栈应用前端

### 第四阶段: 项目实战 (第11-12周)

#### 模块8: 综合项目

- [ ] 完整 AI Agent 应用开发
- [ ] 部署上线
- [ ] 简历项目包装
